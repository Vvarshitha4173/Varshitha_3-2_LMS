Git Pull
Meaning:

Pull is used to download the latest changes from a remote repository (GitHub) to your local system.

Command:
git pull

Use:

Update local code

Get changes made by others

One-line (Exam):

Git pull fetches and merges changes from the remote repository to the local repository.

2. Git Push
Meaning:

Push is used to upload your local commits to the remote repository (GitHub).

Command:
git push

Use:

Save code on GitHub

Share code with team

One-line (Exam):

Git push sends local repository changes to the remote repository.

Fork creates a personal copy of a repository in your GitHub account.

4. Git Clone
Meaning:

Clone creates a local copy of a remote repository on your computer.

Command:
git clone <repository-url>

Use:

Download full project with history

Start working locally

One-line (Exam):

Git clone copies a remote repository to the local machine.
